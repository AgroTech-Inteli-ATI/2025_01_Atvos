# Wiframes do sistema

 &emsp; Os wireframes são representações visuais de baixa/média fidelidade que servem para planejar e estruturar a interface do usuário antes da implementação do sistema. No projeto da Atvos, os wireframes são fundamentais para garantir que todas as interações do usuário estejam claramente definidas, organizadas e funcionais. Para o desenvolvimento do projeto, os wireframes foram criados utilizando a ferramenta Figma disponível [aqui](https://www.figma.com/design/j9dFuF6BBFgzURRwLAV25b/WireFrameAtvos?node-id=0-1&t=RKB6ubjnpEJSFegp-1)

**Tela de Login** 

&emsp; A tela de Login é o ponto de entrada para os usuários acessarem o sistema Atvos. Ela foi projetada para ser de fácil acesso, solicitando credenciais básicas para autenticação.

**Tela de Cadastro** 

 &emsp; A tela de Cadastro permite que novos usuários criem uma conta no sistema Atvos. Ela coleta as informações básicas necessárias para o registro e garante que as senhas atendam aos requisitos de segurança.

**Tela de Dashboard** 

&emsp; O Dashboard é a tela principal do sistema, oferecendo uma visão consolidada e resumida das métricas mais importantes relacionadas à gestão de viagens. Ele é projetado para fornecer insights rápidos e permitir que os usuários monitorem o desempenho geral.

**Tela de Lista de Viagem** 

 &emsp; A tela de Lista de Viagem permite aos usuários visualizar e gerenciar todas as viagens registradas no sistema. Ela oferece funcionalidades de busca e paginação para facilitar a navegação e a localização de informações específicas.

&emsp; Por fim, os wireframes do sistema estabelecem uma base sólida para a interface do usuário. Desde a telas de Login e Cadastro até a visão consolidada de métricas no Dashboard e a gestão detalhada na Lista de Viagens, cada tela foi projetada para garantir uma experiência de usuário intuitiva e organizada. Essa abordagem estruturada é necessária para o sucesso da implementação do sistema, assegurando que todas as interações e funcionalidades atendam às necessidades dos usuários de forma clara e objetiva.
